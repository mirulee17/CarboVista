Biomass Estimation System

Web-Based
Tree Biomass Estimation

An interactive geospatial system for estimating aboveground biomass and carbon stock using Sentinel-2 satellite imagery and machine learning models.

Project Overview

This web application allows users to define an Area of Interest (AOI) using a bounding box on satellite imagery and estimate aboveground biomass based on Sentinel-2 data for environmental and forestry analysis.
